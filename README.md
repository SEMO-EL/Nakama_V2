# Nakama Store — Two Dragons Clean Version

This version intentionally removes:
- the 10-sword collection
- the weak replacement SVGs
- all 3D/model-viewer logic

It focuses only on:
- Black Dragon
- White Dragon
- the real uploaded product/detail pictures
- a clean Black vs White Dragon landing page

## Structure

```txt
nakama_two_dragons_clean/
├── index.html
├── styles.css
├── script.js
└── assets/images/
```

## Open locally

You can simply open `index.html` because there is no 3D model loading in this version.

For best testing, use Live Server or:

```bash
python -m http.server 8000
```
